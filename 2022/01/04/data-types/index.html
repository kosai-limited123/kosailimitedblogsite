Data Types

By mdrahat on January 4, 2022January 4, 2022

What is Data type?

Data type is a classification of value which specifies what kind of data is used or being used. The classification is important because not every mathematical calculation is done with the same type of data. It differ based on the operations are being done.

Example

Some commonly used data types are:

  • Integer
  • Floating point

But in programing there are more types of data are used like:

  • Character
  • Double
  • String
  • Boolean
  • List
  • Stack
  • Queue
  • Vector etc.

Classification

Based on implementation and use two types are considered . They are Abstract Data type (ADT) and Concrete Data type (CDT). And also there are normal data types used in programming.

Whys data types are important?

As mentioned before not every operation is done with the same data types. That’s why in programming data are classified. They have their own size on storage. They are used according to their need in program.

Let’s say string is a set of character . But the size of string is 4 bytes where a character is 1 byte. So to store ‘A’ it is efficient to use character data type.

This may vary according to use of the data.

Another example to say is that, suppose you are adding two values one is integer and one is floating . A C complier will give you a result .

What is the data type of the result?

2 + 2.5 = ?

We will say 4.5. But it depends on the data type where the result is stored.

If it is integer then result = 4.

If it is floating point the result = 4.5. And that’s because two different types of data is being used in operation but the storage is floating point.
