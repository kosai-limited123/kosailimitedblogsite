What is Data type?<\/h2>\n\n\n\n
<\/div>\n\n\n\n

Data type is a classification of value which specifies what kind of data is used or being used. The classification is important because not every mathematical calculation is done with the same type of data. It differ based on the operations are being done.<\/p>\n\n\n\n

      Classification <\/h3>\n\n\n\n
      <\/div>\n\n\n\n

      Based on implementation and use two types are considered . They are Abstract Data type (ADT) <\/a>and Concrete Data type (CDT)<\/a>. And also there are normal data types used in programming.<\/p>\n\n\n\n

      Whys data types are important?<\/h3>\n\n\n\n
      <\/div>\n\n\n\n

      As mentioned before not every operation is done with the same data types. That’s why in programming data are classified. They have their own size on storage. They are used according to their need in program. <\/p>\n\n\n\n

      Let’s say string is a set of character . But the size of string is 4 bytes where a character is 1 byte. So to store ‘A’ it is efficient to use character data type.<\/p>\n\n\n\n

      This may vary according to use of the data.<\/p>\n\n\n\n

      Another example to say is that, suppose you are adding two values one is integer and one is floating . A C complier will give you a result .<\/p>\n\n\n\n

      What is the data type of the result?<\/p>\n\n\n\n

      2 + 2.5 = ?<\/p>\n\n\n\n

      We will say 4.5. But it depends on the data type where the result is stored.<\/p>\n\n\n\n

      If it is integer then result = 4.<\/p>\n\n\n\n

      If it is floating point the result = 4.5. And that’s because two different types of data is being used in operation but the storage is floating point.<\/p>\n","protected":false},"excerpt":{"rendered":"
