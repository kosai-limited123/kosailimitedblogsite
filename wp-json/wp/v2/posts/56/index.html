What is ADT?<\/h3>\n\n\n\n
<\/div>\n\n\n\n

The data type whose behaviour is defined by a set of values and a set of operations. A user can easily manipulate data and the operations of them. The operations are pre defined . It is not necessary to know how the operations are done but they are used in many operations. <\/p>\n\n\n\n

<\/div>\n\n\n\n

Example<\/h3>\n\n\n\n
<\/div>\n\n\n\n

Let’s imagine you have brought a drill machine. The purpose was to put nails making holes that’s it. You are doing your job everyday. But have you ever tried to know how it is functioning? What parts are used ?<\/p>\n\n\n\n

No!! Because you don’t need to now how they are functioning you are placing nails and pressing the buttons.<\/p>\n\n\n\n

That’s what exactly a ADT is. The functionality are defined , you just have to use data and do the operation as you like.<\/p>\n\n\n\n
