Introduction<\/h3>\n\n\n\n
<\/div>\n\n\n\n

An array is a kind of data structure consisting of a collection of elements or data . And it is important to declare of a fixed size of the array every time we declare it . So, it becomes a problem . Hence vector a sequence container that can change size according to the data size . This container is a object that contains the data in a dynamic array . So, it has the ability to resize itself. In array after inserting or deleting it needs to be reallocated or the data reallocates their position. So, you can avoid the problem of fixing a size for array. Vector uses iteration to traverse the elements.<\/p>\n\n\n\n
